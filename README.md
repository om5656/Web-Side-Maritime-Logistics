# Simple Interest Calculator

## Description
This project is a simple interest calculator implemented using a Bash script. 
It calculates simple interest based on user input.

## Features
- Accepts Principal amount
- Accepts Rate of interest
- Accepts Time period
- Calculates Simple Interest using the formula:

Simple Interest = (Principal × Rate × Time) / 100

## Usage
Run the script using:

bash simple-interest.sh

Then enter:
- Principal
- Rate of Interest
- Time Period

The script will display the calculated Simple Interest.
